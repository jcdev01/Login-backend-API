# 🔐 Login API

API simples de **login e cadastro de usuários** desenvolvida com **Java e Spring Boot**.

## 🚀 Tecnologias

* Java 21
* Spring Boot
* Spring Web
* Spring Data JPA
* Spring Security
* MySQL / PostgreSQL
* Maven

## 📡 Endpoints

**Cadastro**

```
POST /auth/register
```

**Login**

```
POST /auth/login
```

## ▶️ Como executar

Clonar o repositório:

```
git clone https://github.com/seu-usuario/login-api.git
```

Executar o projeto:

```
mvn spring-boot:run
```

A API será iniciada em:

```
http://localhost:8080
```
